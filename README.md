# Portfólio Pessoal 
Portfólio profissional desenvolvido com **Next.js** e **TypeScript**, criado para centralizar minha identidade como Desenvolvedora Front-End e apresentar meus projetos.

[Acesse o projeto ao vivo](https://potfolio-talita-nextjs.vercel.app/)
## Sobre o projeto
Aplicação construída com foco em performance, usabilidade e identidade visual própria. O portfólio reúne projetos, habilidades técnicas e informações de contato em uma interface responsiva e moderna.

## Tecnologias

- **Next.js**: framework React com SSR e otimização de performance
- **TypeScript**: tipagem estática para maior confiabilidade do código
- **React.js**: componentização da interface
- **CSS / Tailwind CSS**: estilização responsiva


## Como rodar localmente
### Clone o repositório
`git clone https://github.com/TalitaMoul/portfolio-talita-nextjs.git`

### Acesse a pasta
`cd portfolio-talita-nextjs`

### Instale as dependências
`npm install`

### Inicie o servidor de desenvolvimento
`npm run dev`<br>

Acesse *http://localhost:3000* no navegador.
